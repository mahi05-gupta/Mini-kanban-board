Mini Kanban Board

A simple Kanban Board built using HTML, CSS, and JavaScript to manage tasks across different stages.

Features

- Create tasks (title & description)
- Three columns: Pending, In Progress, Completed
- Move tasks between columns
- Edit and delete tasks
- Responsive design
  
Tech Stack

- HTML
- CSS
- JavaScript

How to Run

1. Clone the repo
2. Open "index.html" in browser
